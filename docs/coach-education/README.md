## Hurtig reference — til brug i coaching-sessioner

### Vis videoer fra:
- **Top Tennis Training YouTube** — pro-demonstrationer af specifikke slag
- **TopCourt** (topcourt.com) — ATP/WTA spillere demonstrerer egne slag
- **SportsEdTV** (gratis) — Grand Slam-niveau coach-demonstrationer

### Vigtigste mentale framework:
1. Gallweys Self 2-princip (Inner Game of Tennis)
2. Pre-point rutinen — 4 trin, 5–8 sekunder, altid det samme
3. Gilberts 80%-regel (Winning Ugly) — identificér modstanderens svaghed

### Aggressiv baseliner-taktik:
- Inside-Out Forehand (IOF) mønster → alternér med Inside-In
- 2 slag cross-court → 1 slag DTL
- Approach shot: DTL backhand eller crosscourt forehand → net
